== Installation ==

This section describes how to install the WP WRITUP plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/wpwritup` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
